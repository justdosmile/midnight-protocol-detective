# AGENTS.md

## Проект

«Три коротких звонка» — статическая русскоязычная детективная игра на React, TypeScript и Vite.

## Структура

- `src/components/` — интерфейс следователя и игровые задачи.
- `src/data/` — публичные материалы дела без открытого решения.
- `src/state/` — прогресс, сохранение и открытие глав.
- `src/utils/` и `src/audio/` — фильтры, шифрование и локальная музыка.
- `public/assets/` — только локальные растровые WebP-изображения.
- `scripts/` — проверка спойлеров и генерация зашифрованного финала.
- `src/test/` и `e2e/` — unit- и браузерные проверки.
- `.private/` — внутренняя логика дела и тестовые ответы; папка всегда игнорируется Git.

## Команды

- `npm run lint` — ESLint и проверка исходников.
- `npm run typecheck` — TypeScript.
- `npm run test` — unit-тесты.
- `npm run build` — production-сборка.
- `npm run test:e2e:prod` — полное прохождение production-версии.
- `npm run check:spoilers` — поиск открытых фрагментов финального объяснения.

## Ассеты

В runtime запрещены внешние изображения, CDN и сетевые запросы. Содержательные изображения должны быть локальными WebP с понятным `alt`. Не добавляйте SVG-иллюстрации. Новые художественные кадры создавайте в той же реалистичной стилистике дома «Вереск» и проверяйте на целевых экранах.

## Безопасность решения

Никогда не добавляйте `.private/`, открытое имя преступника, варианты финального ответа, `.env`, токены или cookie в Git. Production принимает имя только через успешную расшифровку локального пакета. Обходы и отладочные маршруты запрещены.

## Готовность

Изменение готово после успешных `lint`, `typecheck`, `test`, `build`, `check:spoilers` и `test:e2e:prod`, визуальной проверки экранов, полного игрового прогона и проверки отсутствия 404, ошибок консоли и внешних runtime-запросов.
