# AGENTS.md

## Проект

«Полуночный протокол» — статическая русскоязычная детективная игра на React, TypeScript и Vite.

## Структура

- `src/components/` — интерфейс терминала и игровые механики.
- `src/data/` — публичные материалы дела без открытого решения.
- `src/state/` — модель прогресса, сохранение и правила открытия глав.
- `src/utils/` и `src/audio/` — фильтры, криптография и локальный Web Audio.
- `public/assets/` — только локальные изображения и графика.
- `scripts/` — воспроизводимая генерация служебных игровых ресурсов.
- `src/test/` и `e2e/` — unit- и браузерные проверки.
- `.private/` — внутренняя логика дела и тестовые секреты; папка всегда игнорируется Git.

## Команды

- `npm run dev` — локальная разработка.
- `npm run lint` — ESLint и проверка отсутствия запрещённых маркеров.
- `npm run typecheck` — строгая проверка TypeScript.
- `npm run test` — unit-тесты Vitest.
- `npm run build` — production-сборка.
- `npm run preview` — локальный просмотр сборки.
- `npm run test:e2e` — Playwright-прохождение основных сценариев.

## Правила ассетов

Все используемые игрой ресурсы должны храниться в репозитории. Запрещены внешние CDN, удалённые изображения и runtime-запросы. SVG обязаны иметь `viewBox`, осмысленные `title`/`desc` и проверяться на целевых разрешениях.

## Безопасность решения

Никогда не добавляйте `.private/`, открытое имя преступника, финальное разоблачение, тестовые ответы, `.env`, токены или cookie в Git. Production принимает финальное имя только через успешную AES-GCM-расшифровку bundle; обходы и debug routes запрещены.

## Критерии готовности

Изменение готово только после успешных `lint`, `typecheck`, `test`, `build`, `test:e2e`, визуальной проверки 1366×768 / 1440×900 / 1920×1080, полного игрового прогона и проверки отсутствия 404/ошибок консоли.
